Tutorial de Iniciação a Programação na Linguagem C

INTRODUÇÃO

ALGORITMOS

Um algoritmo pode ser definido como uma sequência de passos que visam atingir um objetivo bem definido.

Uma vez concebida uma solução algoritmica para um problema, esta pode ser traduzida para qualquer linguagem de programação e ser agregada das funcionalidades disponíveis nos diversos ambientes, chamado processo de codificação.

Nem sempre todas as ações previstas devem ser executadas. Tal circunstância sugere que um determinado conjunto de ações pode ser evitado, selecionando conforme o resultado de uma determinada condição.

Constroe-se assim uma estrutura seletiva através de um teste condicional que permita ou não que o fluxo de execução passe por um determinado conjunto de ações.

Quando nos deparamos com a inviabilidade da estrutura de seleção para a verificação do êxito da ação, precisamos repetir um mesmo trecho do algomitmo enquanto a condição não esteja satisfeita, construindo uma estrutura de repetição.

TIPOS DE DADOS

TIPOS PRIMITIVOS:

Os dados manipulados pelos algoritimos podem ser dos seguintes tipos:

Tipo Inteiro: toda e qualquer informação numérica que pertença ao conjunto de números inteiros relativos (negativa, nula ou positiva).

Sintaxe: int ID1, ID2;

Tipo Real: toda e qualquer informação numérica que pertença ao conjunto dos números reais (negativa, nula ou positiva).

Sintaxe: float ID1, ID2;

Tipo Caracter: toda e qualquer informação composta por um conjunto de caracteres alfanuméricos: numéricos (0,...,9), alfabéticos (a,...z, A,...,Z) e especiais (#, ?, !, @,...).

Sintaxe: char ID1, ID2;

Tipo Lógico: toda e qualquer situação que possa assumir duas situações (biestável): V (verdade) e F (falsidade).

CONSTANTES:

É um dado constante que não sofre nenhuma variação ao decorrer do tempo.
Ex: Pi

VARIÁVEL:

Um dado é classificado como variável quando tem a possibilidade de ser alterado em algum instante do decorrer do tempo.
Ex: cotação do dólar

IDENTIFICADORES:

Para guardar os dados, precisamos de identificadores, que servem de rótulo para dados variáveis ou constantes, e que para usá-los é necessária a declaração, na qual associamos o identificador a um dos tipos primitivos válidos.

EXPRESSÕES ARITMÉTICAS:

Aquelas cujos operadores são aritméticos e cujos os operandos são constantes ou variáveis do tipo numérico (inteiro ou real).

Os operadores mais utilizados são: + (soma), - (subtração), * (multiplicação), / (divisão) e % (módulo).

EXPRESSÕES LÓGICAS:

Aquelas cujos operadores são lógicos ou relacionais:

Operações relacionais: realizam comparações entre dois valores de mesmo tipo primitivo.

Operadores: == (igualdade), != (diferente de), > (menor que), < (maior que), <= (menor ou igual que), >= (maior ou igual que).

Operações lógicos: são três conectivos básicos pra formação de novas proposições lógicas compostas a partir de outras proposições lógicas simples.

Operadores: && (E lógico), || (OU lógico), ! (Negação lógica).


COMANDO DE ATRIBUIÇÃO

Permite fornecer um valor a uma variável, em que o tipo de dado deve ser compatível com o tipo de variável.

Sintaxe: variável = expressão;

COMENTÁRIOS

Os comentários não se destinam a ser interpretados pelo compilador ou por qualquer componente do processo de desenvolvimento. São simplesmente ignorados pelo compilador, e o programa executável não terá qualquer sinal deles.

Um comentário em C ´qualquer conjunto de caracteres compreendido entre os sinais de /* e */. 


	/* AUTOR: LUIZ LIMA*/


	/*************************
	* Programa 1             *
	* Autor: Luiz Lima       *
	* Data: 05/09/2018       *
	*************************/

COMANDO DE ENTRADA E SAÍDA

Entrada de dados: Para que o algoritmo possa receber os dados de que necessita, precisamos adotar um comando de entrada de dados denominado leia, cuja finalidade é atribuir o dado a ser fornecido à variável identificada.

O comando "leia" é representado pela função scanf.

Saída de dados: Para que o algoritmo possa mostrar os dados que calculou, como resposta ao problema que resolveu, adotamos um comando de saída de dados denominado escreva, cuja finalidade é exibir o conteúdo da variável identificada.

O comando "escreva" é representado pela função printf.

BLOCOS

  • Conjunto de ações, com uma função definida;
  • Serve também para definir os limites nos quais as variáveis declaradas em seu inteior são conhecidas;
  • Delimitadores: "Início" e "Fim".

ESTRUTURAS DE CONTROLE

O algoritimo deve ser regido por um fluxo de execução.

Através das estruturas básicas de controle do fluxo de execução - sequenciação, seleção, repetição - e da combinação delas, poderemos criar algoritmos para solucionar nossos problemas.

Ordem da estrutura:
  1. Início (configurada pela utilização de abertura de chaves);
  2. Declaração das variáveis;
  3. Instruções;
  4. Fim (configurado pela utilização de fechamento de chaves).

Notas:
  • Para termos acesso a essas e outras funções de entrada/saída devemos incluir nos programas o arquivo stdio.h;
  • Em C, um programa começa com a função main();
  • A declaração da variável tem que ser sempre realizada antes da sua utilização e antes de qualquer instrução;
  • Uma variável pode ser automaticamente iniciada quando se faz a sua declaração;
  • Em C é possível atribuir o mesmo valor a várias variáveis;
  • O código a executar é colocado entre {};
  • Um bloco é formado por qualquer conjunto de instruções entre {};
  • Cada instrução deve ser seguida por um ponto e vírgula(;);
  • As strings em C são delimitadas pelo caractere aspas ";
  • O formato de escrita de um inteiro na função printf é %i;
  • O formato de escrita de um número real na função printf é %f;
  • O formato de escrita de um caracter na função printf é %c;
  • Para ler uma variável do tipo int, float ou char utilizando a função scanf é preciso preceder cada variável de um &;

ESTRUTURA SEQUENCIAL

  • O conjunto de ações primitivas será executado em uma sequência linear de cima para baixo e da esquerda para a direita, ou seja, da mesma forma que foram escritas.
  • As ações serão seguidas por um ponto e vírgula (;).
Exemplo utilizando a função printf e scanf: 

	#include <.studio.h.>
	
	main()
	{
		int n1, n2;

		printf ("Introduza dois números:");
		scanf ("%i%i", &n1, &n2);
		printf ("O resultado de %i + %i = %i\n, n1, n2, n1+n2");		
	}

    SWITCH:

    • A fim de eliminar conjuntos extensos de instruções if-else encadeadas, C coloca à disposição a instrução switch;
    • Ao contrário do if, que permite definir bandas de valores (x>=10 && x<=20), o switch só admite valores predefinidos do tipo char, int e long;
    • Depois de a instrução swith verificar qual o case correspondente à expressão avaliada, inicia a execução na instrução associada a esse case, executando em seguida todas as instruções seguites em cascata (inclusive as que pertençam a outros cases) até terminar o switch ou até ser encontrada a instrução break;
    • A instrução break permite parar a execução de um conjunto de instruções dentro de um switch, continuando o programa na instrução seguinte ao switch.
      • .

	#include <.stdio.h.>
	
	main()
	{
		char Est_civil;

		printf ("Qual o estado civil:");

			case 'd': 
			case 'D': printf ("Divorciado\n"); break;
